# Power-BI-sales-analysis-dashboard

## Project Overview

This project focuses on building an interactive Power BI Dashboard to analyze overall Sales Performance and evaluate Outlet Performance across different business locations. The dashboard provides clear visual insights that help compare outlet efficiency, monitor key business metrics, and support data-driven decision-making.

## Features Implemented
KPI Cards to display important business metrics
Column Charts for analyzing sales trends and comparisons
Donut Charts for category-wise sales distribution
Gauge Charts for measuring performance against targets
Comparative analysis of outlet performances
Interactive dashboard design for better business insights

## Key Insights Generated
Overall sales performance analysis
Comparison of performance across multiple outlets
Identification of high-performing and low-performing outlets
Target vs actual performance tracking
Better understanding of business growth patterns

## Tools & Technologies Used
Power BI
Data Cleaning & Transformation
Data Visualization
Business Intelligence Reporting
Dashboard Design
Project Goal

The objective of this project was to transform raw sales data into an interactive dashboard that helps stakeholders understand business performance and make better strategic decisions.
